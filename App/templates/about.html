About FpsoPedia

This page was created to help seafarers on board DP Shuttle tankers operating in Brazilian oil fields to get the required information regarding FPSO's position reference systems, its status and parameters, cargo data for cargo stowage planning and all other required information regarding connection patterns, flushing water transfer and backflushing requirements.

Each vessel maintains its own database in different formats and sizes. The most recent information regarding last cargo's density, API and temp, status of PRS system, and connection pattern is passed from vessel to vessel via email or social media. Administrators' intention is to get all information in one place and make it easy to access and update.

Every user is invited to participate in this process. Everyone can feel free to create the account and add and update the FPSO's information.

Administrators of this website do not store any private data and have no access to users' passwords. The login process is required only to add, edit, or delete FPSO and is created only in order to secure the web page database. If you don't want to create the account, you can still use the page and access all the required information.


In case of any questions or suggestions you can reach us at info@naviidevelopment.atthost24.pl
